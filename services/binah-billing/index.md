# Binah Billing - API Documentation

Welcome to the Binah Billing API documentation.

## Overview

The Binah Billing Service handles subscription management, metering, and billing for the Binelek platform.

## API Reference

For detailed API documentation, see the [API Reference](api/index.md).

## Architecture

The service is built with:
- **.NET 8.0** - Modern C# web API
- **ASP.NET Core** - Web framework
- **Prometheus** - Metrics and monitoring
