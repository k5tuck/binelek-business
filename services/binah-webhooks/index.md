# Binah Webhooks - API Documentation

Welcome to the Binah Webhooks API documentation.

## Overview

The Binah Webhooks Service provides webhook management and delivery for event notifications.

## API Reference

For detailed API documentation, see the [API Reference](api/index.md).

## Architecture

The service is built with:
- **.NET 8.0** - Modern C# web API
- **ASP.NET Core** - Web framework
- **Prometheus** - Metrics and monitoring
